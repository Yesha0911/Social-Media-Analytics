# 📊 Social Media Analytics

## Overview
This project is developed using Python, Pandas, and Matplotlib to analyze social media data and visualize user engagement through graphs.

## Features
- Read data from CSV file
- Display user statistics
- Analyze likes and engagement
- Generate bar chart using Matplotlib
- Menu-driven Python application

## Technologies Used
- Python
- Pandas
- Matplotlib

## Files
- main.py
- sma.csv
- Figure_1.png

## Output
The project reads social media data and displays insights using graphs
